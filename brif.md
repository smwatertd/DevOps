задание следующее:

1) написать приложение на языке php
2) в качестве веб-сервера для запуска приложения обязательно используем nginx
3) в качестве субд - MySQL или PostgreSQL (SQLite и прочие легковесные СУБД - нельзя)

приложение разворачиваем 2 способами:
1) на виртуальной машине без использования контейнеров
2) на виртуальной машине с использованием контейнеров (docker compose) либо, если есть соответствующие навыки, в кластере Kubernetes

суть приложения - простая система-опросник или система тестирование (вроде университетского moodle) на любую тему, авторизация пользователей не нужна, но нужно иметь возможность просматривать результаты тестирования/опроса в интерфейсе приложения

в обоих вариантах разворота нужно подумать о безопасности приложения:  пароли не должны быть зашиты в коде приложения, "лишние" порты не должны быть открыты во внешнюю сеть и т.д.

отчет по заданию можете прислать в текстовом формате, только максимально подробно опишите, что, как и где у вас развернуто (где расположены файлы приложения, какие конфигурационные файлы влияют на его работу и т.д.)
